therian

  • Last Published: 2014-11-25 |
  • Version: 0.4-SNAPSHOT

therian provides typesafe transformations of Java object graphs.

Features and priorities:

  • Deep awareness of Java Unified EL for smooth integration into the JEE stack
  • Strong typing
  • Fluent API
  • Fully extensible
